# 日報システム
（https://github.com/raynoduomua/dailyreport）

自社で使用している日報システムを私なりに作成したものです。

1.受講生が日々の学習内容を報告する

2.講師が日々の講義内容を報告、受講生の評価、テスト結果を入力する。

3.クライアント（企業様）は受講生の学習内容、講師の評価を確認できる。

4.管理者は、受講生、クライアント等のアカウント作成や、講座名の作成ができる
